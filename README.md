# CLI Task Manager

## Project Vision
A simple CLI-based task manager to practice Python fundamentals and simulate real-world development using Agile Scrum.

---

## Features
- Add task
- List tasks
- Mark task as done
- Delete task
- Edit task
- Save/load tasks
